# List UF codes used by DATASUS downloads

Returns a character vector of the 27 Brazilian state abbreviations
accepted by DATASUS file naming conventions.

## Usage

``` r
datasus_ufs()
```

## Value

A character vector of length 27.

## Examples

``` r
datasus_ufs()
#>  [1] "AC" "AL" "AM" "AP" "BA" "CE" "DF" "ES" "GO" "MA" "MG" "MS" "MT" "PA" "PB"
#> [16] "PE" "PI" "PR" "RJ" "RN" "RO" "RR" "RS" "SC" "SE" "SP" "TO"
```
